【提示】136人均无感染情况！松江首个隔离观察点所有人员解除观察

/

( 2020-02-09)

/
/
/

松江区新闻办说，2月8日下午5点30分开始，松江区首个疫情防控隔离观察点的所有人员解除隔离，自1月25日大年初一开始，到今天正月十五元宵节，经过14天集中隔离观察后，136名人员确认均无相关感染。详见↓

/

 

/

14天观察，136人均无相关感染

/

2月8日，松江区疫情防控隔离观察点内136名人员在确认无相关感染情况后，开始陆续解除隔离。

/


/

下午4点30分，松江区疾控中心学校卫生科科长姚春霞手提几份餐食，按照隔离人员离开的先后时间，挨门挨户开始送餐，“5点30分第一户人家就要走了，所以今天想提前把饭给他们送去，不要吃得太着急。”

/

 

/

姚春霞说道，之后，她还要协助其他医护人员准备隔离人员离开的各类事宜，“他们要先签一份健康观察解除告知单，再测一次体温，确认没问题了，就可以离开了。”

/


/

据松江区疾病预防控制中心的吴医生透露，自1月24日上海宣布启动重大突发公共卫生事件一级响应机制后，松江区便设立了首个疫情防控重点地区来沪人员的隔离点，每一位从疫情防控重点地区来沪的人员都将留在这里接受为期14天的观察，截至1月26日上午11点，便有136人入住隔离点，“136位可以离开的隔离人员都有不同的离开时间，因为我们要严格遵照隔离14天的要求。”吴医生说道。

/

“除了谢谢，还是谢谢”

/

下午5点30分，黄女士带着一家人从电梯下来，完成体温监测，签订健康观察解除告知单后，黄女士对着所有的医护人员连连说着感谢。

/



/

1月25日，刘女士和丈夫带着3岁的女儿一同来到了松江区的隔离观察点，由于女儿年纪太小，“她待了两天就会闹一闹，小孩子不懂，为什么自己不能出去。”更令刘女士无奈的是，女儿太过依赖奶奶，常常会想家中奶奶，“除了视频，或者做点其他事情转移注意力之外，其他也没有什么办法。”

/

 

/

除了情绪问题之外，刚到隔离点不过几天，女儿便出现了咳嗽的情况，那时的刘女士心急如焚，“这种情况下，有点感冒发烧都很让人担心。”刘女士说道，幸运的是，隔离观察点的医护人员在得知刘女士一家的情况后，及时上门替女儿做了检查，确认只是普通咳嗽，并开了一些咳嗽药水后，不过一天，女儿的情况便得到了好转。

/

 

/

除此之外，一日三餐的供应，情绪上的疏导，隔离点医护人员无微不至的关怀让刘女士和丈夫感动不已，“14天的时间里，他们无微不至的照顾我们，真是太谢谢她们了。”离开前，刘女士紧紧握着医护人员的双手，连连感谢。

/

熬过了春节，终于可以过一个元宵节

/

吴医生说，在隔离观察点的人员今年春节可以算是“熬”过来的，整整14天，没有同家人真正团聚过。而解除隔离的日子，正逢2020年元宵当天。

/


/

瞿女士一家在昨晚10点从隔离观察点离开，她记得25号当天刚来到隔离观察点时，“真的觉得很难接受，大家都在团聚，怎么就我家被隔离了起来？特别难受。”瞿女士5岁的儿子频频调皮捣蛋，让原本难熬的瞿女士心里更是烦闷。

/

 

/

但即便如此，面对每天两次的体温监测和医护人员的情绪开导，瞿女士一家也终于慢慢接受需要隔离的事实，“小朋友虽然一直都不是很理解，但是我们大人的心态调整好了，小朋友也能从我们的反应和态度中感觉到这次事情的严重性。”伴随着春节假期的结束，瞿女士知道，只要慢慢等待，总能健康回家。

/
