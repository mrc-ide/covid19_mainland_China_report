2020年2月8日广西新型冠状病毒肺炎疫情情况

/
发布日期：2020-02-09 08:00:04    点击：
39823
    来源：自治区卫生健康委员会
/ /
/
/
//
/
/
/
/
/ /
/
/
/

 

2月8日0-24时，我区共新增新型冠状病毒肺炎确诊病例12例，其中南宁市3例、北海市1例、河池市4例、来宾市4例。新增治愈出院病例1例（梧州市1例），新增死亡病例1例（河池市都安县1例）。

目前全区累计报告确诊病例195例，累计出院病例18例。现有重症病例6例（南宁市1例、桂林市1例、北海市1例、贵港市1例、河池市2例），危重病例7例（南宁市1例、桂林市1例、防城港市2例、玉林市1例、河池市2例），累计死亡病例1例（河池市都安县1例）。

累计确诊病例中，南宁市35例、柳州市20例、桂林市29例、梧州市5例、北海市33例、防城港市16例、钦州市8例、贵港市7例、玉林市9例、百色市3例、贺州市4例、河池市19例、来宾市7例。

目前累计追踪到密切接触者9701人，正在接受医学观察3985人。

新型冠状病毒感染人群，除了主要通过呼吸道飞沬传播外，还可经过人体粘膜、皮肤等接触传播。专家建议，双手在以下7种情形没有进行有效洗手之前，千万不要碰触脸、眼睛及口鼻，以免病毒入侵而感染疾病。1.吃东西前。难免双手会接触食物，容易将病毒吃下去。2.如厕后。双手可能沾染尿液或粪便，或是碰触马桶盖、开关门、冲水时也可能沾染病菌。3.擤鼻涕后。双手可能沾染大量病菌。4.戴口罩前。双手可能会接触病菌，容易接触口鼻。5.就医前后。医院环境容易存在病菌，就医前后洗手，可避免将细菌带入医院，或把病菌带回家。6.处理食材前后。准备烹饪的食材之前与之后，应及时洗手。7.接触动物后。避免动物身上的细菌、病毒入侵。
