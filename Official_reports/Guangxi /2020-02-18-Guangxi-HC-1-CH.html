2020年2月17日广西新型冠状病毒肺炎疫情情况

/
发布日期：2020-02-18 08:00:03    来源：自治区卫生健康委员会
/ /
/
/
//
/
/
/
/
/ /
/
/
/

2月17日0-24时，我区共新增新型冠状病毒肺炎确诊病例4例（南宁市3例、北海市1例）；新增疑似病例21例；新增治愈出院病例7例（南宁市2例、桂林市2例、北海市1例、贵港市1例、玉林市1例）；无新增死亡病例。

目前全区累计报告确诊病例242例，累计出院病例58例，累计死亡病例2例（河池市1例、北海市1例），现有确诊病例182例，均在院治疗，其中重症病例3例（南宁市1例、防城港市1例、河池市1例），危重病例7例（南宁市1例、桂林市2例、北海市1例、防城港市2例、河池市1例）。现有疑似病例93例。

累计确诊病例中，南宁市54例、柳州市24例、桂林市31例、梧州市5例、北海市44例、防城港市18例、钦州市8例、贵港市8例、玉林市10例、百色市3例、贺州市4例、河池市22例、来宾市11例。

目前累计追踪到密切接触者15106人，正在接受医学观察1719人。

疫情持续，营养重要。一般人群（病人除外）如何合理膳食，才能有效增强抵抗力、进一步提高预防新型冠状病毒感染的能力？专家建议，一要食物多样，谷类为主。每天膳食有谷薯类、蔬菜水果类、畜禽鱼蛋奶类、大豆坚果类等食物。二要多吃蔬果、奶类、大豆。做到餐餐有蔬菜，天天吃水果，多选深色蔬果，不以果汁代替鲜果；吃各种各样的奶及其制品，特别是酸奶，相当于每天液态奶300克；常吃豆制品，适量吃坚果。三要适量吃鱼、禽、蛋、瘦肉。适量摄入，少吃肥肉、烟熏和腌制肉制品，坚决杜绝食用野生动物。四要少盐少油，控糖限酒。清淡饮食，少吃高盐和油炸食品；足量饮水，多饮白开水和茶水，成年人每天1500-1700毫升，不喝或少喝含糖饮料；如成人饮酒，男性一天饮用酒的酒精量不超过25克，女性不超过15克。五要吃动平衡，健康体重。天天运动，减少久坐时间，每小时起来动一动，保持健康体重。六要杜绝浪费，兴新食尚。学会阅读食品标签，按需备餐，提倡分餐和使用公筷、公勺；选择新鲜、安全的食物和适宜的烹调方式；食物制备生熟分开、熟食二次加热要热透。
