2020年2月7日广西新型冠状病毒感染的肺炎疫情情况

/
发布日期：2020-02-08 08:00:36    点击：
32506
    来源：自治区卫生健康委员会
/ /
/
/
//
/
/
/
/
/ /
/
/
/


2月7日0-24时，我区共新增新型冠状病毒感染的肺炎确诊病例11例，其中南宁市1例、桂林市1例、北海市1例、防城港市1例、钦州市1例、玉林市1例、河池市4例、来宾市1例。新增疑似病例123例，现有疑似病例295例。

目前全区累计报告确诊病例183例，累计出院病例17例。现有重症病例7例（南宁市2例、桂林市1例、防城港市3例、贵港市1例），危重病例8例（南宁市1例、桂林市1例、防城港市2例、玉林市1例、河池市3例），无死亡病例。

累计确诊病例中，南宁市32例、柳州市20例、桂林市29例、梧州市5例、北海市32例、防城港市16例、钦州市8例、贵港市7例、玉林市9例、百色市3例、贺州市4例、河池市15例、来宾市3例。

目前累计追踪到密切接触者8877例，尚在接受医学观察4072例。

今天是农历正月十五，在疫情存续情况下，为了你、我、他的健康，每个家庭和个人都应自觉扛起防控责任，进一步阻断疫情的传播和扩散。专家建议，要保持家中室内空气流通，多开窗户多通风，适时增减衣服，避免受凉感冒；要保持双手卫生，饭前便后、打喷嚏、咳嗽和清洁鼻子，以及外出归来应洗手，决不随地吐痰；外出要戴口罩，尽量不串门访客或少去公共场所，避免聚餐活动；要合理安排好饮食，多饮水和摄入足够维生素，宜多食富含优质蛋白、糖类及微量元素的食物，如瘦肉、禽蛋、新鲜蔬菜、水果等；要积极开展个人健身体育锻炼，使身体气血畅通，增强体质；要保持充分休息睡眠，不要通宵熬夜玩牌、打电子游戏机等，否则会降低身体免疫力而容易患病。

祝福大家度过一个安定祥和健康美丽的元宵佳节！
