2020年2月23日广西新型冠状病毒肺炎疫情情况

/
发布日期：2020-02-24 08:00:00    来源：自治区卫生健康委员会
/ /
/
/
//
/
/
/
/
/ /
/
/
/

2月23日0-24时，我区新增新型冠状病毒肺炎确诊病例2例（河池市2例）；新增疑似病例4例；新增治愈出院病例3例（南宁市1例、柳州市1例、钦州市1例）；无新增死亡病例。

目前全区累计报告确诊病例251例，累计出院病例107例，累计死亡病例2例（河池市1例、北海市1例），现有确诊病例142例，均在院治疗，其中重症病例6例（南宁市1例、桂林市1例、防城港市2例、河池市2例），危重病例5例（南宁市1例、北海市1例、防城港市2例、河池市1例）；现有疑似病例46例。

累计确诊病例中，南宁市55例、柳州市24例、桂林市32例、梧州市5例、北海市44例、防城港市19例、钦州市8例、贵港市8例、玉林市11例、百色市3例、贺州市4例、河池市27例、来宾市11例。

本日追踪到密切接触者22人，现有1137人正在接受医学观察。

疫情存续期间，非发热类患者去医院就诊时，应如何避免交叉感染的风险？专家建议，患者和陪同家属应全程佩戴口罩；去医院前，可事先通过网络或电话预约拟就诊医疗机构，熟悉医院科室布局和步骤流程，减少就诊时间和避免走错发热门诊就诊；前往医院首选私家车或出租车，尽量避免乘坐公共交通工具；在医院就医时，尽可能避免与有发热、咳嗽或打喷嚏等症状的人密切接触，人与人之间尽可能保持一定距离（至少一米），如果手接触到医院门把手、门帘、医生白大衣等医院物品后，可使用备用的便携含酒精成分免洗洗手液来消毒，若不能及时洗手，不要接触口、鼻、眼，打喷嚏或咳嗽时，用纸巾遮住口鼻；患者和陪同家属或者陪同家属就医归来后，应该彻底清洁消毒，进一步降低病毒在家中传播感染的风险。
