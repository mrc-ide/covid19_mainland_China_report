/ 天津首位确诊患者今日出院 累计治愈病例已达10人/ /

//
/
发布日期：2020-02-11 22:48
/
来源：委办公室
/
浏览次数:/
/

振奋人心

/

天津再传捷报

/

2月11日，天津市海河医院再有2名患者经过医护人员的精心救治，达到了国家卫生健康委《新型冠状病毒感染的肺炎诊疗方案》解除隔离治疗的标准，顺利康复出院，其中一位出院患者是本市确诊的第一例新型冠状病毒肺炎患者。目前，天津治愈病例已经达到了10例。

/

/

本市首例新冠肺炎确诊病例，也是出院的首名重症患者。该患者于1月21日被确诊为我市首例新冠肺炎确诊病例，基础合并糖尿病，冠心病冠脉支架，脑梗死，高血压病。经过红区一线医护人员及市级医疗救治专家20多天的全力救治，精心呵护，患者的各项生理指标逐渐恢复正常，体温连续多日恢复正常。复查胸部CT片显示肺部炎症已逐渐吸收，连续两次呼吸道病毒核酸检测结果均呈阴性。经市专家会诊符合国家卫生健康委确定的新型冠状病毒感染的肺炎确诊病例解除隔离和出院标准，于今天出院。

/

/

目前，海河医院院内的患者正在逐渐好转。重症患者的出院，给奋战在疫情防控一线的广大医务人员以极大的鼓舞，同时也为重症患者的医疗救治积累了宝贵经验。
