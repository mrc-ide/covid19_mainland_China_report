/ 2020年2月7日天津市新型冠状病毒感染的肺炎疫情情况/ /

//
/

/
来源：委办公室
/
浏览次数:/ / / /
/
//

2020年2月6日0-24时，天津市报告新型冠状病毒感染的肺炎新增确诊病例10例，新增危重型病例3例，新增重型病例8例，新增死亡病例0例，新增出院病例0例。其中：

北辰区为首次报告确诊病例。其它各区新增确诊病例中宝坻区6例、西青区1例、东丽区1例、滨海新区1例。

截至2月6日24时，天津市累计报告新型冠状病毒感染的肺炎确诊病例79例，危重型病例7例，重型病例28例，死亡病例1例，出院病例2例。其中：

确诊病例中，宝坻区21例、河东区10例、河北区10例、和平区6例、河西区4例、宁河区4例、东丽区4例、西青区4例、南开区3例、滨海新区3例、红桥区2例、津南区1例、北辰区1例、外地来津6例；出院病例中，河东区1例、红桥区1例。

目前追踪到密切接触者599人，已解除医学观察62人，尚有516人正在接受医学观察。

天津市卫生健康委员会

2020年2月7日
