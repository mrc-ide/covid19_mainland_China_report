/ 天津再有10名新冠肺炎患者出院 截至今日中午累计出院75人！/ /

//
/
发布日期：2020-02-23 16:57
/
来源：委办公室
/
浏览次数:/ / / /
/
//
//

今天中午，天津又有10名新冠肺炎患者治愈出院。截至2月23日中午，天津市累计治愈出院人数达到75人。自新冠肺炎疫情发生以来，天津市累计治愈出院人数首次超过住院人数。

昨天，一名10岁的患儿已经达到出院标准。但医护人员考虑到孩子需要母亲照料，就让他继续留在母亲身边。今天，孩子的母亲经检查最终判定，符合出院标准，母子俩一起走出了隔离病区。出院时，孩子还带着医护人员送给他的玩具和书。

据孩子母亲介绍，目前，自己的丈夫和女儿仍在接受治疗，希望两人积极配合医护人员，尽早回家团聚。
