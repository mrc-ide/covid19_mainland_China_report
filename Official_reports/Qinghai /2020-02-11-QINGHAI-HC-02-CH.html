青海医疗队与当地成功救治的3例新冠肺炎患者出院

/

/

2020年02月11日　来源: 省指挥部办公室（健康宣传组）

/
/

 2月10日14时06分，武汉市新洲区 43岁的新冠肺炎患者沈某，顺利地从新洲区人民医院出院。据了解，这是青海医疗队和当地医疗部门联手成功救治的首例新型冠状病毒感染的肺炎危重患者。当天上午11时02分，48岁的患者周某和28岁的患者罗某也顺利地出从新洲区中医院出院。

//

 1月26日沈某因发热、胸闷住进新洲区人民医院感染内科，1月31日转入ICU。青海专家组成员孙斌、侯明、佘小斌、孙晓林和新洲区人民医院医生曾兴宏、邱世明、王晓芬、陈明成立联合医疗组对其进行紧急救治，2月2日病人脱离危重状态。此后在专家组及医护团队全力救治和精心护理下，沈某病情持续改善，呼吸道症状好转，肺部影像学显示病灶明显吸收，连续4天体温正常，各项化验指标均正常。2月6日、9日核酸检测均为阴性，经专家组评估，沈某符合国家卫生健康委新冠肺炎确诊病例解除隔离和出院标准，2月10日顺利出院。

//

 青海医疗队专家侯明说：“沈先生能顺利康复，得益于青海医疗队和新洲区人民医院的重症团队的共同合作，精心治疗，再加上沈先生意志力比较顽强，配合治疗也非常好，所以经过11天的治疗，目前已经达到了临床出院的标准，祝贺他。这是我们从1月31日改建完成病区入住病人后第一个痊愈出院的患者，是我们建科以来重症患者的第一例出院患者，恢复不错。”

//

 青海医疗队孙晓林护士长说：“沈先生今天出院，大家都很高兴，治疗期间，沈先生很配合医疗队的护理工作，希望沈先生在家里继续加强康复训练，沈先生加油，武汉加油，中国加油，我们大家一起加油！”

//

 青海医疗队专家佘小斌说：“沈先生是我们青海医疗队来新洲组建重症医学科之后的第一批患者，也是我们第一个重症痊愈出院的患者，在此向他表示祝贺。沈先生的出院，是大家一起努力的结果，也是我们所有人的期望，我相信，有了这第一例，就会有更多的第二例、第三例……，我想武汉会好起来的，就像沈先生一样会好起来的！”

//

 走出住院大楼，沈某连声感谢党和政府，不停地向医护人员道谢：“青海的医疗队对我们很关心，早上中午晚上都会去看我们，从技术上治疗，从心理上鼓励我不要怕，调整好心态，嘱咐我什么事都没有，该吃吃，该睡睡，医生让怎么做我就怎么做，现在恢复得很好，都正常了”。
