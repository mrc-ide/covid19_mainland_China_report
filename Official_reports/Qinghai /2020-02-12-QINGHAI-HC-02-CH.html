喜报！青海省又有四例新冠肺炎患者康复出院

/

/

2020年02月12日　来源: 省指挥部医疗救治组

/
/

 2月12日下午14时30分，我省治愈的四例新冠肺炎患者从省第四人民医院出院。这是我省第三批出院患者，据了解，省第四人民医院共收住新冠肺炎患者15例，累计出院7例。至此，青海省确诊的18例患者已有9例出院，医疗救治工作取得了阶段性成效。

//

 此次出院的四名患者，陈某某，系我省首例确诊患者，1月23日入院，25日确诊。祁某某，1月27日入院并于当日确诊。马某某，1月26日在西宁市回族医院留观后转入省第四人民医院并于当日确诊。韩某某，2月3日确诊后从门源县人民医院转运至省第四人民医院治疗。

//

 2月11日，根据国家《新型冠状病毒感染的肺炎诊疗方案（试行第五版 修正版）》出院标准，经省级多学科专家评估，符合出院标准。出院后，将按照居住地医疗卫生机构安排，再完成14天单人单间隔离。

//

 四名患者入院以来，我省按照国家诊疗方案，坚持“集中病例、集中专家、集中资源、集中救治”的救治原则，调集省内呼吸、重症、院感、影像、检验、药学、中医等学科最精锐力量，实行中西医结合、一人一方案、一人一团队，科学有序开展医疗救治，积极联系北京协和医院、中日友好医院的国家级专家线上会诊，不断调整优化治疗方案。省第四人民医院每日组织专家评价病情全力救治，并通过心理疏导、营养支持、人文关爱、医患沟通、健康教育等方式，鼓励四名患者积极配合治疗，增强战胜疾病的信心和勇气，为早日治愈出院赢得了时间。

//

 省第四人民医院院长郗爱旗说，作为全省唯一一所省级传染病专科医院，疫情发生以来，全院职工始终把患者的生命安全和身体健康放在第一位，齐心应对、沉着应战、迎难而上，冲锋在疫情防治阻击战的最前沿，永不退缩、毫无怨言，努力用自己的“辛苦指数”换取群众的“安全指数”。隔离病区医护人员舍小家为大家，主动放弃休假机会，一天四班倒昼夜坚守在工作岗位。这当中，大部分是80后或90后，都是上有老下有小的年纪，但为了战胜疫情，他们把孩子托付给老人或朋友，很多人已20多天没有回家。为了治愈患者，他们不惧被感染的风险，穿着密封的防护服，日夜穿梭在病床之间，尽全力救治患者，给予患者最精心的治疗和最精细的呵护。这期间，为节约防护物资，大家不敢多喝水、多吃饭，只怕上厕所耽误时间浪费防护物资。正是有了他们的默默付出，日夜坚守，换回一个个患者康复，更加坚定了我们打赢疫情防控阻击战的信心和决心。我为他们点赞。
